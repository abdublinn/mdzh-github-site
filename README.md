# МДЖ сайт (статический MVP)

Готовый статический сайт по ТЗ `МДЖ_ТЗ_на_разработку_сайта_v2.md`.

## Что уже реализовано

- Основные разделы: главная, решения, процесс и стоимость, о нас, коммерческая мебель, блог, сервис, контакты.
- Квиз 5 шагов с валидацией контакта, прогрессом, кнопкой назад и подбором похожих кейсов.
- Фильтры каталога решений с синхронизацией в URL (`?type=...&segment=...&q=...`).
- Плавающие CTA-кнопки, cookie-баннер, sticky-хедер с hide-on-scroll.
- Скрытые QR-страницы: `care/kitchen`, `care/wardrobe`, `care/kids`, `care/bathroom`, `care/living`.
- Базовые SEO-файлы: `robots.txt`, `sitemap.xml`.

## Публикация на GitHub Pages

1. Создать новый репозиторий и загрузить содержимое этой папки.
2. Включить GitHub Pages: **Settings -> Pages -> Deploy from a branch**.
3. Выбрать ветку `main`, папку `/ (root)`.
4. После публикации:
   - заменить `https://example.github.io` в `sitemap.xml` и `robots.txt` на ваш домен;
   - при необходимости прописать кастомный домен.

## Что нужно заполнить перед продом

- Живые фото/видео, реальные тексты кейсов, диапазоны цен.
- Реальные ссылки Telegram/телефон в `data.js`.
- Подключение CRM, GA4, Метрики, reCAPTCHA на серверной стороне.
